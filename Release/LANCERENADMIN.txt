Si une erreur du format suivant arrive "Error 0xc000007b pour Windows 7, 8.1, 10"
Suivez ces etapes :
1 veuillez mettre a jour votre driver de carte graphique. 
2 mettre a jour la version Microsoft .NET framework
3 desinstaller le jeu 
4 eteindre et rallumer votre PC
5 reinstaller le jeu en executant l'installeur avec les droits administrateur
